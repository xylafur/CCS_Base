XDS Emulation Software Package (EMUPack) Update
 

Release

Name: EMU18_M11

Build Number: 8.0.903.2

Date: 2018-11-20

New Features and Improvement

  • Linux host support Cortex-A53 and Cortex-R5F ETM trace for AM65x device (DBGTRC-4338)
  • Linux host support for CoreSignt STM and CPTracer2 trace for AM65x device (DBGTRC-4340)
  • Linux host support for cache analysis use cases for AM65x device (DBGTRC-3740)
  • TPIU pin trace support for Cortex-A53 and Cortex-R5F ETM trace for AM65x device (DBGTRC-4342)

Bug fixes

  • ET noise on Agama LPs with JTAG jumpers mounted (DBGTRC-4353)
  • Crash in AET on connect to all MSP430s on 8.3.0.00001-emu-02 (DBGTRC-4349)
  • Profile Clock is not available for ARM9 of OMAP L138 when using CCSv8.x (DBGTRC-4334)
  • XDS200 fails to work on macOS Mojave (DBGTRC-4328)
  • TWP data source hanging on 560V2 STM decode (DBGTRC-4318)
  • Using control-c in GDB to stop a program executing on MSP430 causes gdb agent to crash (DBGTRC-4304)
  • Count Event and Profile Count support missing for ARM968 on Vayu class devices (DBGTRC-4027)
  • DRA74x (J6Plus) Trace does not have a proper device ID to enable trace (DBGTRC-3912)

Known Issues

  • Port stune to Linux and macOS (DBGTRC-4406)
  • AM65x Cortex-R5 trace produces much less trace data with XDS560 Pro than with ETB (DBGTRC-4385)
  • Auto-selection of serial port does not work on OSX for MSP432 Launchpad (DBGTRC-4372)
  • SMP Debug: Cannot run past a HW breakpoint (DBGTRC-4331)
  • CCS variables view displays only lower 16 bits of 32 bit variable allocated to XAR register (DBGTRC-4309)
  • EnergyTrace automatic Debug Probe detection does not work (DBGTRC-4156)
  • Expressions view does not correctly display value of float variable in FPU registers (DBGTRC-4145)
  • XDS110 pod fails to provide power to CC32xx devices during test connection (DBGTRC-3998)
  • Group core fails to properly control the ARM/DSP cores of an OMAPL device (DBGTRC-3954)
  • XDS GDB Agent does not work with XDS100/XDS110 for Multi-core devices (DBGTRC-3927)
  • SWO Trace and TM4C129/MSP432E: lower Sample Intervals cause JTAG instability (DBGTRC-3897)
  • ETB PC trace does not produce expected results for 66AK2Hxx with NDK examples (DBGTRC-3593)
  • PC Trace should not display 560 V2 Trace as a viable Transport Type (DBGTRC-3588)
  • XDS GDB agent fails to display a suitable error message in non-supported targets (DBGTRC-3399)

IDE Compatibility

  • CCS IDE: This update has been tested with CCSv8.2 and up.
  • 3rd party IDE: This update has been tested with IAR EWARM 8.30.2.18209 and Keil uVision v5.26.2.0 on Windows 10 64-bit environment.
